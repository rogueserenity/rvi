# Security Policy

## Supported Versions

Only the latest code on the `main` branch receives security fixes. There are no versioned releases at this time.

## Reporting a Vulnerability

Please use [GitHub's private vulnerability reporting](https://github.com/rogueserenity/rvi/security/advisories/new) to report security issues. This keeps the report confidential until a fix is available.

You can expect an acknowledgement within a few days and a fix or status update within 90 days, depending on severity.
